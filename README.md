🏠 **Zari House – Responsive Website**

Zari House is a modern and clean front-end website designed using HTML, CSS, Bootstrap, and JavaScript.  
It focuses on responsive layout, structured design, and smooth user experience across all devices.

---

✨ **Features:**
- Modern and clean 
- Fully responsive layout (mobile, tablet, desktop)  
- Image carousel / slider section  
- Well-structured sections (Home, Collection, About, Contact)  
- Smooth navigation and user-friendly UI  
- Built using Bootstrap grid system  

---

🛠️ **Technologies Used:**
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript  

---

📌 **Project Purpose:**
This project was created for front-end development practice.  
It demonstrates responsive design, layout structuring, and Bootstrap-based UI building for real-world website layouts.
